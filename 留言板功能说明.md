# 留言板功能说明

## 功能概述

为中医古籍典藏网站新增了留言板功能，访客可以在此留下对网站的建议、感受和交流。

## 功能特性

### 📝 留言提交
- **姓名**：必填，显示在留言卡片上
- **联系方式**：选填，可以填写邮箱或微信号
- **留言内容**：必填，支持多行文本

### 💬 留言展示
- 留言按时间倒序排列（最新的在最前）
- 每条留言显示：
  - 头像（姓名首字母）
  - 姓名和联系方式
  - 留言内容
  - 发布时间（智能显示：刚刚、X分钟前、X小时前、X天前）

### 🎨 界面设计
- 中国风配色，与网站整体风格一致
- 模态框设计，不影响主页浏览
- 响应式布局，支持移动端
- 流畅的动画效果

### 💾 数据存储
- 使用 localStorage 本地存储
- 数据存储在用户浏览器中
- 每个浏览器独立存储，不共享

### 🛡️ 安全防护
- XSS 防护：自动转义 HTML 标签
- 表单验证：必填项检查
- 数据格式：JSON 结构化存储

### 🔧 管理功能
- **删除留言**：双击留言卡片可删除（管理员功能）
- **初始示例**：首次访问自动添加4条示例留言

## 使用方法

### 访客使用

1. **打开留言板**
   - 点击页面底部的 "📝 留言板" 链接
   - 或直接访问页面后点击留言板按钮

2. **填写留言**
   - 输入姓名（必填）
   - 输入联系方式（选填）
   - 输入留言内容（必填）
   - 点击"提交留言"按钮

3. **查看留言**
   - 留言提交后自动显示在列表中
   - 可以查看其他访客的留言

### 管理员使用

**删除不当留言：**
- 双击要删除的留言卡片
- 在确认对话框中点击"确定"

## 技术实现

### 数据结构

```javascript
{
  id: 1234567890,              // 时间戳作为唯一ID
  name: "访客姓名",             // 留言者姓名
  contact: "联系方式",          // 联系方式（可选）
  message: "留言内容",          // 留言文本
  timestamp: "2025-03-17T..."  // ISO格式时间戳
}
```

### 存储位置

- **键名**：`tcm_guestbook`
- **位置**：`localStorage`
- **格式**：JSON数组

### 示例数据

```javascript
[
  {
    "id": 1710662400000,
    "name": "中医学子",
    "contact": "tcm_student@example.com",
    "message": "非常感谢提供这么好的资源！",
    "timestamp": "2025-03-17T08:00:00.000Z"
  }
]
```

## 功能限制

### 当前限制
- ❌ 留言不跨浏览器同步
- ❌ 留言不跨设备同步
- ❌ 没有后端存储
- ❌ 没有用户认证系统
- ❌ 没有留言审核机制

### 适用场景
- ✅ 个人学习和测试
- ✅ 本地部署使用
- ✅ 小范围团队使用

## 清除数据

如需清除所有留言：

```javascript
// 在浏览器控制台执行
localStorage.removeItem('tcm_guestbook');
```

或者：

```javascript
// 清除所有localStorage数据
localStorage.clear();
```

## 导出数据

导出留言数据（在浏览器控制台执行）：

```javascript
// 获取留言数据
const messages = localStorage.getItem('tcm_guestbook');
console.log(messages);

// 下载为JSON文件
const blob = new Blob([messages], {type: 'application/json'});
const url = URL.createObjectURL(blob);
const a = document.createElement('a');
a.href = url;
a.download = 'guestbook_' + Date.now() + '.json';
a.click();
```

## 导入数据

导入留言数据（在浏览器控制台执行）：

```javascript
// 假设 jsonData 是导入的JSON字符串
const jsonData = '[...]'; // 你的JSON数据
localStorage.setItem('tcm_guestbook', jsonData);
location.reload(); // 刷新页面
```

## 常见问题

### Q: 为什么我看不到其他人的留言？
**A:** 留言存储在浏览器本地，每个浏览器的数据是独立的。

### Q: 换了浏览器留言会丢失吗？
**A:** 是的，localStorage 数据存储在特定浏览器中。

### Q: 如何实现多人共享留言？
**A:** 需要使用后端服务器或第三方评论系统。

### Q: 留言数据安全吗？
**A:** localStorage 数据存储在用户本地，相对安全，但可被用户自行修改或删除。

### Q: 可以限制留言长度吗？
**A:** 可以在表单中添加 `maxlength` 属性限制字符数。
