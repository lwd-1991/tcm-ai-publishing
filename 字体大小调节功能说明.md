# 字体大小调节功能说明

## 功能概述

TXT编辑器新增了字体大小调节滑动条，用户可以根据自己的阅读习惯自由调整编辑器中文字的显示大小。

## 新增功能

### 🎚️ 字体大小滑动条
- **位置**：状态栏左侧
- **范围**：10px - 24px
- **步进**：1px
- **默认**：14px

### 💾 自动保存偏好
- 用户调整后自动保存到浏览器
- 下次打开自动应用上次的设置
- 使用 localStorage 持久化存储

### 🔄 重置功能
- 一键重置到默认大小（14px）
- 点击"重置"按钮即可
- 显示成功提示

## 使用方法

### 调整字体大小

**方法一：拖动滑块**
1. 找到状态栏的"字体大小"控制区
2. 拖动滑块左右移动
3. 实时查看编辑器字体变化
4. 释放鼠标，设置自动保存

**方法二：点击轨道**
1. 点击滑动条轨道的任意位置
2. 滑块跳转到点击位置
3. 字体大小立即更新

### 重置字体大小

1. 点击"重置"按钮
2. 字体大小恢复到 14px
3. 显示成功提示

### 查看当前大小

- 滑动条右侧实时显示当前字体大小
- 格式：`14px`

## 技术实现

### HTML 结构

```html
<div class="font-size-control">
    <span class="font-size-label">字体大小:</span>
    <input 
        type="range" 
        class="font-size-slider" 
        id="fontSizeSlider" 
        min="10" 
        max="24" 
        value="14" 
        step="1"
    >
    <span class="font-size-value" id="fontSizeValue">14px</span>
    <button class="font-size-reset" onclick="resetFontSize()">重置</button>
</div>
```

### JavaScript 核心代码

**初始化函数：**
```javascript
function initFontSizeControl() {
    const slider = document.getElementById('fontSizeSlider');
    const valueDisplay = document.getElementById('fontSizeValue');
    
    // 加载保存的字体大小
    const savedFontSize = localStorage.getItem('tcm_editor_font_size');
    if (savedFontSize) {
        const fontSize = parseInt(savedFontSize);
        slider.value = fontSize;
        valueDisplay.textContent = fontSize + 'px';
        if (editor) {
            editor.updateOptions({ fontSize: fontSize });
        }
    }
    
    // 监听滑动事件
    slider.addEventListener('input', function() {
        const fontSize = parseInt(this.value);
        valueDisplay.textContent = fontSize + 'px';
        
        if (editor) {
            editor.updateOptions({ fontSize: fontSize });
        }
        
        // 保存到 localStorage
        localStorage.setItem('tcm_editor_font_size', fontSize);
    });
}
```

**重置函数：**
```javascript
function resetFontSize() {
    const defaultSize = 14;
    const slider = document.getElementById('fontSizeSlider');
    const valueDisplay = document.getElementById('fontSizeValue');
    
    slider.value = defaultSize;
    valueDisplay.textContent = defaultSize + 'px';
    
    if (editor) {
        editor.updateOptions({ fontSize: defaultSize });
    }
    
    localStorage.setItem('tcm_editor_font_size', defaultSize);
    showToast('字体大小已重置为 ' + defaultSize + 'px', 'success');
}
```

**Monaco Editor 更新：**
```javascript
editor.updateOptions({ fontSize: 14 });
```

## 字体大小范围

### 可选范围
- **最小**：10px（适合小屏幕或需要查看更多内容）
- **默认**：14px（推荐，适合大多数用户）
- **最大**：24px（适合视力不佳或大屏幕）

### 推荐设置

| 场景 | 推荐大小 | 说明 |
|------|---------|------|
| 笔记本电脑 | 14px | 默认大小，平衡舒适度 |
| 大屏显示器 | 16-18px | 更舒适的阅读体验 |
| 小屏幕 | 12px | 查看更多内容 |
| 视力不佳 | 20-24px | 更大更清晰 |
| 演示/投影 | 22-24px | 远距离可见 |

## 数据存储

### localStorage 配置

**键名：** `tcm_editor_font_size`
**数据类型：** 字符串（数字）
**示例值：** `"14"`, `"16"`, `"20"`

### 读取设置

```javascript
const savedFontSize = localStorage.getItem('tcm_editor_font_size');
console.log('当前字体大小:', savedFontSize + 'px');
```

### 修改设置

```javascript
// 设置为 16px
localStorage.setItem('tcm_editor_font_size', '16');
location.reload();
```

### 清除设置

```javascript
// 清除字体大小设置
localStorage.removeItem('tcm_editor_font_size');
location.reload();
```

## 常见问题

### Q: 字体大小设置会保存吗？
**A:** 会的，调整后自动保存到浏览器，下次打开自动应用。

### Q: 不同浏览器的设置会同步吗？
**A:** 不会，localStorage 数据存储在特定浏览器中。

### Q: 可以设置更大或更小的字体吗？
**A:** 当前范围是 10-24px。如需修改，可以调整 HTML 中的 min/max 属性。

### Q: 重置按钮会清除文件内容吗？
**A:** 不会，只会重置字体大小，不影响文件内容。

### Q: 字体大小影响打印吗？
**A:** 不影响，打印时使用浏览器的打印设置。
