# 中医古籍典藏 - 使用说明

## 重要提示

由于浏览器安全策略限制，本项目**不能**直接双击HTML文件打开使用。

必须通过**本地HTTP服务器**运行才能正常使用所有功能。

## 启动方法

### 方法一：使用 Python（推荐）

1. 打开终端（Terminal）
2. 进入项目目录：
   ```bash
   cd /Volumes/PSSD/TCM-Ancient-Books
   ```
3. 启动HTTP服务器：
   ```bash
   python3 -m http.server 8000
   ```
4. 在浏览器中访问：
   ```
   http://localhost:8000/index.html
   ```

### 方法二：使用 Node.js

1. 安装 http-server（仅需一次）：
   ```bash
   npm install -g http-server
   ```
2. 进入项目目录并启动：
   ```bash
   cd /Volumes/PSSD/TCM-Ancient-Books
   http-server -p 8000
   ```
3. 在浏览器中访问：
   ```
   http://localhost:8000/index.html
   ```

### 方法三：使用 PHP

```bash
cd /Volumes/PSSD/TCM-Ancient-Books
php -S localhost:8000
```

## 功能说明

### 主页功能
- **搜索古籍**：在搜索框输入关键词查找古籍
- **分类筛选**：按医学门类筛选古籍
- **编辑按钮**：点击书籍卡片上的"✏️ 编辑"按钮在编辑器中打开

### TXT编辑器功能
- **打开文件**：打开本地TXT文件
- **新建文件**：创建空白文档
- **保存文件**：保存修改到本地
- **另存为**：保存为新文件
- **转码**：字符编码转换（UTF-8、GBK、GB2312等）

### 快捷键
- `Ctrl/Cmd + O`：打开文件
- `Ctrl/Cmd + S`：保存文件
- `Ctrl/Cmd + N`：新建文件

## 常见问题

### Q: 为什么点击"编辑"按钮显示"Failed to fetch"？
**A:** 这是因为没有通过HTTP服务器运行。请按照上述方法启动本地服务器。

### Q: 如何停止HTTP服务器？
**A:** 在终端按 `Ctrl + C` 即可停止。

### Q: 可以修改端口号吗？
**A:** 可以。将命令中的 `8000` 改为其他端口号即可，如 `python3 -m http.server 3000`

### Q: 编辑后的文件保存在哪里？
**A:** 保存的文件会下载到浏览器的默认下载目录。
