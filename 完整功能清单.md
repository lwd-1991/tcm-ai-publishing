## 🚀 快速开始

### 启动方法

**方法一：使用启动脚本**
```bash
./启动服务器.sh
```

**方法二：手动启动**
```bash
cd /Volumes/PSSD/TCM-Ancient-Books
python3 -m http.server 8000
```

**访问地址：**
- 主页：http://localhost:8000/index.html
- 编辑器：http://localhost:8000/editor.html
