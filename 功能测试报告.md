1. **File System Access API**
   - Safari 和 Firefox 不支持

   - 不影响基本功能

2. **编码转换**
   - 浏览器原生支持有限
   - 当前为演示版本
   - 建议集成 encoding.js 库

3. **访问计数器**
   - 基于 localStorage
   - 不跨浏览器同步
   - 适合个人使用
