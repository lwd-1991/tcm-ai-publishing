- 当前星期
- 实时时钟（时分秒）

**更新频率：**
- 每秒自动更新
- 无需手动刷新

**显示格式：**
```

```

## 技术实现

### 日期时间功能

**核心代码：**
```javascript
function updateDateTime() {
    const now = new Date();
    
    // 格式化日期
    const year = now.getFullYear();
    const month = String(now.getMonth() + 1).padStart(2, '0');
    const day = String(now.getDate()).padStart(2, '0');
    const dateStr = `${year}年${month}月${day}日`;
    
    // 格式化时间和星期
    const weekdays = ['星期日', '星期一', '星期二', '星期三', '星期四', '星期五', '星期六'];
    const weekday = weekdays[now.getDay()];
    const hours = String(now.getHours()).padStart(2, '0');
    const minutes = String(now.getMinutes()).padStart(2, '0');
    const seconds = String(now.getSeconds()).padStart(2, '0');
    const timeStr = `${weekday} ${hours}:${minutes}:${seconds}`;
    
    // 更新显示
    document.getElementById('currentDate').textContent = dateStr;
    document.getElementById('currentTime').textContent = timeStr;
}

// 初始化并每秒更新
updateDateTime();
setInterval(updateDateTime, 1000);
```

### 访问计数器功能

**数据存储：**
- 存储位置：localStorage
- 键名：`tcm_visit_count`
- 数据类型：字符串（数字）

**核心代码：**
```javascript
function initVisitCounter() {
    // 获取当前计数
    let count = parseInt(localStorage.getItem('tcm_visit_count') || '0');
    
    // 计数+1
    count++;
    
    // 保存新计数
    localStorage.setItem('tcm_visit_count', count.toString());
    
    // 显示动画
    animateCounter(count);
}

function animateCounter(targetCount) {
    const counterElement = document.getElementById('visitCounter');
    const duration = 1000; // 1秒
    const steps = 30;
    const increment = targetCount / steps;
    let current = 0;
    
    const timer = setInterval(() => {
        current += increment;
        if (current >= targetCount) {
            current = targetCount;
            clearInterval(timer);
        }
        counterElement.textContent = Math.floor(current).toLocaleString();
    }, duration / steps);
}
```

## 使用说明

- 打开网站即可看到实时时钟
- 时间每秒自动更新
- 显示当前系统时间

### 查看访问次数
- 访问量显示在时钟右侧

### 重置计数器

如需重置访问计数，在浏览器控制台执行：

```javascript
// 重置为0
localStorage.setItem('tcm_visit_count', '0');
location.reload();

// 设置为指定数值
localStorage.setItem('tcm_visit_count', '1000');
location.reload();

// 删除计数数据
localStorage.removeItem('tcm_visit_count');
location.reload();
```

### 导出计数数据

```javascript
// 获取当前计数
const count = localStorage.getItem('tcm_visit_count');
console.log('当前访问量:', count);
```

## 常见问题

### Q: 为什么访问量一直增加？
**A:** 每次刷新页面都会计数+1，这是正常行为。

### Q: 不同浏览器的访问量不一样？
**A:** 是的，localStorage 数据存储在特定浏览器中，不同浏览器独立计数。

### Q: 清除浏览器缓存会影响计数吗？
**A:** 会的，清除浏览器数据会重置计数器。

### Q: 时间显示不准确怎么办？
**A:** 时间来自用户设备的系统时间，请检查系统时间设置。

### Q: 可以隐藏日期时间或计数器吗？
**A:** 可以通过 CSS 隐藏：
```css
.datetime-display { display: none; }
.counter-display { display: none; }
```

### Q: 如何修改计数器样式？
**A:** 修改 CSS 中的 `.counter-display` 和 `.counter-number` 类。
