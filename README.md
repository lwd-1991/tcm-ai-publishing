# 中医古籍  tcm-ai-publishing

一个优雅的中医古籍在线阅读和编辑平台，收录700+部中医经典著作。

## ✨ 功能特性

### 📚 古籍典藏
- 收录700+部中医经典著作
- 涵盖本草、伤寒、金匮、内经、方书等50+医学门类
- 支持搜索和分类筛选
- 精美的卡片式展示

### ✏️ TXT编辑器
- **Monaco Editor** - VS Code同款编辑器
- 语法高亮，自动识别文件类型
- 支持打开本地文件、编辑、保存
- 字符编码转换（UTF-8、GBK、GB2312等）
- 快捷键支持（Ctrl+S保存、Ctrl+O打开等）

### 🎨 界面设计
- 中国风配色方案
- 响应式设计，支持移动端
- 流畅的动画效果

## 🚀 快速开始

### 方法一：使用启动脚本（推荐）

```bash
# 双击运行或在终端执行
./启动服务器.sh
```

### 方法二：手动启动

```bash
# 进入项目目录
cd /Volumes/PSSD/TCM-Ancient-Books

# 启动HTTP服务器
python3 -m http.server 8000

# 在浏览器访问
# http://localhost:8000/index.html
```

## ⚠️ 重要提示

**不能直接双击HTML文件打开！**

由于浏览器安全策略限制，必须通过本地HTTP服务器运行才能使用编辑器功能。

## 📖 使用说明

### 主页功能
1. **搜索古籍** - 在搜索框输入关键词
2. **分类筛选** - 点击分类按钮筛选
3. **打开编辑器** - 点击书籍卡片上的"✏️ 编辑"按钮

### 编辑器功能
1. **打开文件** - 点击"打开文件"或按 `Ctrl+O`
2. **编辑内容** - 在编辑器中修改文本
3. **保存文件** - 点击"保存文件"或按 `Ctrl+S`
4. **转码** - 点击"转码"按钮转换字符编码

### 快捷键
- `Ctrl/Cmd + O` - 打开文件
- `Ctrl/Cmd + S` - 保存文件
- `Ctrl/Cmd + N` - 新建文件

## ❓ 常见问题

### Q: 点击"编辑"按钮显示"Failed to fetch"？
**A:** 请确保通过HTTP服务器运行，而不是直接打开HTML文件。

### Q: 如何停止服务器？
**A:** 在终端按 `Ctrl + C`

### Q: 可以修改端口吗？
**A:** 可以。修改启动命令中的端口号，如 `python3 -m http.server 3000`

### Q: 编辑后的文件保存在哪？
**A:** 保存的文件会下载到浏览器的默认下载目录
